# ML Models From Scratch

Implementation of **Linear Regression** and **Logistic Regression** built from scratch using Python and NumPy — without using any ML libraries like scikit-learn.

## Models
- **Linear Regression** — Predicts continuous values using Gradient Descent
- **Logistic Regression** — Predicts binary classes using Sigmoid function
- ** Polynomial Regression** - Helps to predict the values by ploting a Polynomial function 
## Technologies
- Python 3.x
- NumPy

## How to Run
```bash
python Linear_Regression.py
python Logestic_regression.py
python Polynomial_Regression.py
```
